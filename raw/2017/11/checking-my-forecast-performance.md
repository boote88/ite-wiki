> _Because more young adults are becoming unemployed on account of they can't find work. Basically, the problem is this: if you haven't got a job, then you’re outta work! And that means only one thing — unemployment!_

> The Young Ones (1982) “Demolition”

Actually, the latest unemployment rate data tells us it continues to fall as predicted by the [dynamic information equilibrium model](https://informationtransfereconomics.blogspot.com/2017/01/dynamic-equilibrium-presentation.html) (conditional on the absence of shocks/recessions):



![](<media/dynamic equilibrium -unemployment recent data FRBSF- data updates 0.png>)
![](<media/dynamic equilibrium -unemployment recent data FRBSF- data updates 1.png>)
![](<media/dynamic equilibrium -unemployment recent data FRBSF- data updates 2.png>)

The first is the original prediction, the second is a comparison with various forecasts of the FRBSF, and the third is a comparison with two different Fed forecasts.



In trying to be fair to the FRBSF model, I didn't show the data from the [before I made the graph](https://informationtransfereconomics.blogspot.com/2017/06/forecast-stability-unemployment-rate.html) as new post-forecast data (in black). However, in these versions of the graph I take all of the data from after [the original forecast](http://informationtransfereconomics.blogspot.com/2017/01/unemployment-forecasts.html) (in January) as new:



![](<media/dynamic equilibrium -unemployment recent data FRBSF- data updates 1a.png>)
![](<media/dynamic equilibrium -unemployment recent data FRBSF- data updates 2a.png>)

There also don't appear to be any signs of an oncoming shock yet; however the [JOLTS data](https://informationtransfereconomics.blogspot.com/2017/07/jolts-leading-indicators.html) (in particular, hires) appears to be an earlier indication that the unemployment rate — by about 7 months. That is to say, we should see little in the unemployment rate [until the recession is practically upon us](https://informationtransfereconomics.blogspot.com/2017/04/determining-recessions-with-algorithm.html) (although the algorithm can still see it before it is declared or even widely believed to be happening).
