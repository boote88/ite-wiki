I wanted to compare [the bitcoin forecast](https://informationtransfereconomics.blogspot.com/2017/09/bitcoin-fork-and-sp500-forecast-model.html) to the latest data even though I updated it only last week since according to the model, it should move fast (logarithmic decline of -2.6/y). Even including the "news shock" blip (basically noise) from [Jamie Dimon's comments today](https://www.reuters.com/article/us-usa-banks-conference-jpmorgan/jpmorgans-dimon-says-bitcoin-is-a-fraud-idUSKCN1BN2KP), the path is on the forecast track:



![](<media/bitcoin new data updates.png>)

This is a "conditional" forecast -- it depends on whether there is a major shock (positive or negative, but nearly all have been positive so far for bitcoin).
