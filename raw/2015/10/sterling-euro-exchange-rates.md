Simon Wren-Lewis [calculated](http://mainlymacro.blogspot.com/2015/10/is-sterling-overvalued.html) the equilibrium exchange rate between the Euro and Pound back in 2003:

> _When I calculated an equilibrium sterling euro rate in 2003, my estimate was 1.365 E/£._

which corresponds to 0.73 £/€. Which is completely consistent with the [information transfer model result](http://informationtransfereconomics.blogspot.com/2015/06/exchange-rates-and-irrational-markets.html):



![](<media/eu uk exchange rate -SW-L-.png>)

We both expect a return to a slightly lower value (for €/£, a higher value for £/€). In the ITM it's part of the long run trend of the relative size of the two economies. Wren-Lewis's reasoning is a bit more complicated.
