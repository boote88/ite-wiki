# Not bad for five parameters, take two (2015-01-08)

[Original post](../../raw/2015/01/not-bad-for-five-parameters-take-two.md)

Update of the ITM's US macro predictions (originally made in [March 2014](../../raw/2014/03/)). The model uses just five parameters to predict interest rates, inflation, unemployment, NGDP, and RGDP growth.

## Key finding

The monetary base isn't following the assumed counterfactual as well as expected, which means short-term interest rates should be a bit higher than predicted — and they are starting to rise.

## Significance

Five parameters predicting six macroeconomic variables simultaneously, with data broadly tracking predictions made ~10 months prior. The ITM's predictive power is competitive with models using far more parameters (the Fed's FRB/US model uses 29+).
